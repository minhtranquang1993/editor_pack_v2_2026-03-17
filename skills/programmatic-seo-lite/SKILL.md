# Programmatic SEO Lite

Mục tiêu: chuẩn hoá SEO theo cụm thay vì viết rời rạc từng bài.

## Core outputs

1. **Topic Cluster Map**
- Pillar page
- 5-20 supporting pages
- Intent mỗi trang

2. **Internal Link Plan**
- Pillar -> support
- Support -> pillar
- Support liên quan chéo nhau

3. **Schema Checklist**
- Article
- FAQPage (nếu có FAQ)
- LocalBusiness (nếu local service)
- BreadcrumbList (nếu web hỗ trợ)

## Output format

```text
📚 Cluster: {topic}
- Pillar: ...
- Support pages: ...

🔗 Internal links:
- ...

🧩 Schema:
- Article: YES/NO
- FAQPage: YES/NO
- LocalBusiness: YES/NO
- BreadcrumbList: YES/NO
```

## Definition of Done
- Mỗi trang có intent rõ
- Có plan linking tối thiểu 2 chiều
- Có schema checklist để bàn giao implement

Use this skill when:
- Cần tạo nhiều bài SEO cùng lúc (programmatic)
- Xây dựng topic cluster với internal links
- User says: "SEO nhiều bài", "topic cluster", "internal links", "programmatic SEO"
